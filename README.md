Project Description

DoIt is an AI-inspired academic execution system that bridges the gap between goal setting and structured daily action. It first understands a student’s context, constraints, and goals through a guided onboarding flow, and then converts deadlines into prioritized focus sessions using a smart scheduling engine.

Problem Statement

Students often set academic goals but fail to convert them into structured action. They forget deadlines, struggle with prioritization, and feel overwhelmed when multiple tasks overlap.

Most productivity tools are limited to simple to-do lists or basic calendars and do not consider the student’s personal context, available time, or urgency of deadlines.

There is a need for a system that understands the student first and then intelligently schedules work based on urgency and time constraints.

Solution

DoIt combines structured user profiling with deadline-driven scheduling.

A guided onboarding system collects:

Education level

Academic stream

Goals

Available time

Constraints and friction

A smart planner allows users to:

Add subjects

Add tasks with deadlines

Track deadlines visually through a calendar

Automatically prioritize tasks by urgency

Generate a Pomodoro-based focus timeline

This transforms vague academic goals into structured and executable daily plans.

Features
1. Context-Based Profiling

A chat-style onboarding system collects academic identity, goals, constraints, and friction to understand the user’s situation.

2. Subject and Deadline Management

Users can add subjects and tasks (test, assignment, record) with specific deadline dates. The system automatically calculates remaining days.

3. Calendar Visualization

A dynamic monthly calendar displays exact deadline dates and highlights tasks under each day.

4. Automated Work Scheduler

The user selects a time window. The system:

Sorts tasks by nearest deadline

Assigns 25-minute focus sessions
Generates a structured timeline

5. Persistent Data Storage
